# 🌀 台风播报机器人 (Typhoon Alert Bot)

自动抓取中央气象台台风预报，定时推送到 Slack 频道 **#dept-os-cn**。

---

## 运行逻辑

| 步骤 | 说明 |
|------|------|
| **触发** | 每天固定时间自动唤起任务 |
| **抓取** | 用 WebSearch 取最新台风预报（中央气象台官方数据） |
| **判断** | 与上次记录对比，有更新才推送，无更新则跳过 |
| **推送** | 把整理好的格式发到 Slack #dept-os-cn |
| **记录** | 每次运行写一条日志到 `logs/` 文件夹 |

## 推送时间表

- 每天 **08:40** 和 **14:40**（北京时间）
- 对齐中央气象台主报时次（02/08/14/20时），挑工作时段、电脑通常开着的时间
- cron 表达式：`40 8,14 * * *`

## ⚠️ 运行前提

- 电脑**开机**且未休眠
- Claude 桌面应用在**运行**、且已**登录**
- 到点时若电脑关闭，该次会**错过**，不补发

## 数据源

- 中央气象台台风网 https://typhoon.nmc.cn/
- 通过 WebSearch 获取（官方主报每天约 02/08/14/20 时更新）

## 发件人身份说明

> 自动任务运行在沙箱环境，出于安全只能连接白名单内服务。
> Slack 的 webhook (`hooks.slack.com`) 不在白名单内，因此自动任务
> **无法直接驱动独立 bot**。当前推送通过官方 Slack 连接器发出，
> 消息中以「🤖 自动播报」标识，表明为机器自动推送、非个人发送。
