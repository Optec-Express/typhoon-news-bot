# 消息格式模板

每个 emoji 项单独一行，发送到 Slack #dept-os-cn：

```
🤖 自动播报
🌀 第6号台风「蔷薇」(Jangmi)
📅 更新：6/2
📍 位置：北纬26.6° 东经127.7°（冲绳那霸近海）
🧭 路径：北偏东行，擦过冲绳进东海东南部，后转东北趋向日本南部
🌏 受影响国家/地区：
🇨🇳 中国：东海东部 10–12 级大风，浙江沿海留意
🇯🇵 日本：冲绳、本州南部沿海受影响
🔗 新闻详情（带链接）
📡 数据源：中央气象台 · 自动推送
```

## 字段说明

- **标题**：台风编号 + 中文名（可选保留国际名）
- **更新**：抓取日期
- **位置**：经纬度 + 就近地名
- **路径**：移动方向 + 路径预报
- **受影响国家/地区**：按国家分行，附风力/影响提示
- **新闻详情**：当次抓到的官方/权威新闻链接
- **数据源**：固定标注
